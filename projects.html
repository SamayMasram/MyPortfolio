The system functions as a virtual medicine box , allowing users to categorize, organize, and monitor all their prescriptions through a simple, intuitive interface. Its primary function is to provide automated reminders for timely intake and proactive smart notifications that alert users about impending expiry dates. Core features include detailed medicine sorting, highly customizable dose reminders, and reliable expiry date tracking to prevent health risks associated with outdated medicine. Project Overview

Tea Leaf Disease Detection

Technology Stack

Python, Deep Learning, CNN Model

Accuracy

76.06% Validation Accuracy

Tea Disector

This Project focuses on developing a deep learning-based system for the automatic detection and classification of diseases affecting tea leaves, which is crucial for effective intervention and sustainable farming. Traditional disease diagnosis relies on time-consuming and subjective visual inspection by agricultural experts. The core of the system is a Convolutional Neural Network (CNN) model, which has proven highly effective in image classification tasks. The model was trained using a publicly available dataset from Kaggle, titled "Identifying Disease in Tea Leafs." This dataset comprises images of tea leaves belonging to eight distinct categories: Healthy leaves Seven specific disease types: Red Leaf Spot, Algal Leaf Spot, Bird's Eye Spot, Brown Blight, Gray Blight, White Spot, and Anthracnose. The proposed CNN model achieved a peak validation accuracy of approximately 76.06%, demonstrating a consistent learning progression. The system performed robustly in distinguishing between most disease types. However, the study noted ongoing challenges in accurately classifying conditions that are visually very similar, specifically Brown Blight and Gray Blight. The practical utility of this research is realized through a real-time prediction module. This allows users (like farmers or experts) to upload an image of a tea leaf and receive an instant disease diagnosis, thus streamlining the monitoring process for large-scale tea production.

Semi-automatic Embedded System Project

Technology Stack

Arduino Uno, HC-05 Bluetooth Module, Relay Module, L298N Motor Driver

Resources

Drive

Semi-automatic System for Home

This semi-automatic embedded system project utilizes an Arduino Uno as the central microcontroller to manage the remote control of an LED and a DC fan via Bluetooth. The system integrates an HC-05 Bluetooth module to receive commands wirelessly from a user's smartphone or PC. Upon receiving specific single-digit commands (e.g., '1', '2', '3', '4'), the Arduino processes the input. The relay module acts as an electrically operated switch, controlled by the Arduino, to power on or off the high-voltage/current LED circuit. Simultaneously, an L298N motor driver is used to control the speed and direction of the DC fan, although in this context, it will likely be used for simple on/off control of the fan (or to set a single speed). A portable battery powers the entire system, ensuring autonomous operation. The project demonstrates a basic home automation concept where numbers '1' and '2' could correspond to starting/stopping the LED, and '3' and '4' could control the fan motor, providing a foundational platform for more complex, remotely operated embedded applications.
